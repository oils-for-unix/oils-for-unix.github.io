This page is for linking to places where Oils is used, e.g. its packages on Linux Distros. Feel free to edit it!

Warning: Two Tarballs

As of 2024, there is awkwardness related to the Python -> C++ transition. The Python tarball is packaged in more places, but the long term plan is for it to go away.

Eventually, we will only distribute the C++ tarball.

  • The C++ tarball is oils-for-unix-$VERSION.
  • The Python tarball is oil-$VERSION.

Packages

Please package Oils!

New "Oils for Unix" C++ Tarball

  • DNF (Fedora/RHEL): dnf copr enable tkbcopr/oils-for-unix && dnf in oils-for-unix
  • Homebrew: brew install oils-for-unix
  • Nix: pkgs.oils-for-unix
  • Arch User Repository: pacman -U oils-for-unix-<version>-<architecture>.pkg.tar.zst
    • Arch Wiki: https://wiki.archlinux.org/title/Oil_Shell (Outdated)
  • Alpine Linux: echo https://dl-cdn.alpinelinux.org/alpine/edge/testing >> /etc/apk/repositories && apk add oils-for-unix
    • Optionally replace /bin/sh: apk add oils-for-unix-binsh
    • Optionally replace /bin/bash: apk add oils-for-unix-bash

Old "Oil" Tarball (which includes CPython code)

This tarball is going away at some point. If you maintain these, please migrate to the "Oils for Unix" tarball!

  • ~~Homebrew: brew install oil~~
  • ~~Nix: pkgs.oil~~
  • Pacstall (Debian/Ubuntu): pacstall -I oil
  • Void Linux: xbps-install oil
